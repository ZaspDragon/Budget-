# Budget Paycheck Planner

A simple GitHub Pages budget app that lets you:

- enter your paycheck amount
- choose how often you get paid
- add monthly bills
- see your monthly income, yearly income, yearly bills, and leftover budget
- include a yearly savings goal

## Run locally

Just open `index.html` in your browser.

## Upload to GitHub Pages

1. Create a new GitHub repository.
2. Upload `index.html`, `style.css`, `script.js`, and `README.md`.
3. In GitHub, go to **Settings > Pages**.
4. Under **Build and deployment**, choose:
   - **Source:** Deploy from a branch
   - **Branch:** `main` and `/root`
5. Save.
6. Your app will be live on your GitHub Pages link in a minute or two.

## Customize it

You can edit the default bill categories in `script.js` inside `loadState()`.
